# Kato Turnout modules

Kato mimic panel contains 2 shields
* DCC shield to receive and send retroinformation from DCC, BTN and/or LED ![DCC schematic](/Kato/Mimic-Panel/images/Schematic_Mimic-Panel-Shield-DCC_2026-03-28.png?raw=true)
* 12V DC shield to generate the current for the turnouts (up to 4)
