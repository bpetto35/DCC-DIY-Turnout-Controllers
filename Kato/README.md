# Kato Turnout modules

## Kato mimic panel (TCO) contains 2 shields
* DCC shield to receive and send retroinformation from DCC, BTN and/or LED ![DCC schematic](/Kato/Mimic-Panel/images/DDC_Schematic.png?raw=true)
* 12V DC shield to generate the current for the turnouts (up to 4) ![DC signal schematic](/Kato/Mimic-Panel/images/Signal_Schematic.png?raw=true) ![DC power schematic](/Kato/Mimic-Panel/images/PWR_Schematic.png?raw=true)

## DCC section occupancies (up to 8 with 2 shields)
* 4 sections using jumpers to select ![DCC occupancy](/Kato/DCCDetection/images/DCCOccupation_Schematic.png?raw=true)
