# Manage up to 4 DCC section detection

## Key Features
* 1 input DCC connector
  * horizontal terminal block 3.81mm 2 pins
* 4 ouput DCC connector
* current detection per output
* jumper selection to stack 2 shields for 8 outputs
* Send retroinformation to
  * JMRI using the Serial TX/RX
* Use headers 3.81mm for male screw terminal

### Pins
| Pin # | Function |
|---|---|
| D0 | JMRI RX |
| D1 | JMRI TX |
| D2 | section 4 occupancy information |
| D3 | section 8 occupancy information |
| D4 | section 3 occupancy information |
| D5 | section 7 occupancy information |
| D6 | section 2 occupancy information |
| D7 | section 6 occupancy information |
| D8 | section 1 occupancy information |
| D9 | section 5 occupancy information |
